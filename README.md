This repository shows my Reinforcement Learning (RL) work for the Unitree Go2 robot.

I started this project about two months ago and now have results to share.

What I built:

- Fast walking gait — about 5 m/s
- Slow, stable gait — about 0.6 m/s

I also added domain randomization and terrain variations so the robot can handle:

- Flat ground
- Small bumps
- Stairs and uneven surfaces

It stays balanced and walks well on these terrains.

I also added a camera on top of the robot's head.
